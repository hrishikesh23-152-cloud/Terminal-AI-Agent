# Terminal AI Agent (LangGraph)

A terminal-based AI chatbot with tool calling and memory built using LangGraph.

## Features

- LLM powered agent
- Tavily web search tool
- Conversation memory
- Graph-based agent workflow
- CLI chat interface

## Tech Stack

- Node.js
- LangGraph
- LangChain
- Groq API
- Tavily Search

## Run

npm install
node main.js
